# VIM Setup

## Pathogen

1) Create autoload and bundle folders
```
mkdir -p ~/.vim/autoload ~/.vim/bundle
```

2) Download and install Pathogen
```
curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim
```

3) Setup syntax on. On ~/.vimrc add:  

```
:syntax on
```
